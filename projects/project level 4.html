    third level 

                            In this level we learned about the various tags to structurized the webpage :
                        
  • header tag 
    • 
                        This tag is the top most tag in the website , it's hierarchy should be first it tells about the 
                        overall structure of the website. Basically , it is the introductory part of the webpage.

                        
  • main tag 
    • 
                         This is the tag which consists the main components of the website . It contains the primary 
                         content of the web. The main tag is futher divided into  three tag , they are:
                         
      section tag
    
                         Section tag contains the group of the related content .
                         
       article tag
    
                         Article tag is used to present the content like blogs, post as it is in the website .
                         It consists of the elongated description of the webpage .
                         
      aside tag
    
                         This tag is generally used to keep the things which provides additional information which 
                         do not have direct effect on website . 

                         
  • footer tag
    • 
                         This tag is used for finishing the website . It generally appears at the bottom of the page.
                         Here we can keep our contact info, links of social media, certificates , etc.

                         
  • div tag 
    • 
                         This tag is used to contain the other tags which is beneficial for the our CSS work.
                         
  • span tag
    • 
                         This tag is used to generally perform some operations about on certain words or phases.
                         
  •  navigation tag
    • 
                         This tag is used to navigate the things in the website.
